Férias CLT Pro ⚡

Aplicação web de Departamento Pessoal focada em UI/UX e visualização de dados (Data Visualization) em tempo real. Projeto desenvolvido puramente com Vanilla JavaScript, dispensando frameworks externos.

# 🎯 Destaques do Projeto
- **Lógica Complexa:** Motor matemático para cálculo de férias, variáveis e impostos em cascata (Tabela INSS 2024 e IRRF).
- **Raio-X Financeiro:** Gráfico de barras dinâmico construído via manipulação nativa de DOM para ilustrar ganhos vs. retenções.
- **UI Moderna:** Interface com Dark Mode corporativo, Neumorfismo e responsividade via CSS Grid/Flexbox.

#  💻 Stack
* HTML5 Semântico
* CSS3 Avançado
* JavaScript (ES6+)

# 🚀 Execução
O projeto é 100% Client-side. Para testar, basta clonar o repositório e executar o arquivo `index.html` em qualquer navegador.
